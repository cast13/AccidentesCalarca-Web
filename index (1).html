Dashboard de Accidentes en Calarcá 2017-2022

Resumen del Proyecto

Este proyecto analiza la accidentalidad registrada en el municipio de Calarcá entre los años 2017 y 2022, utilizando datos abiertos del portal datos.gov.co. El sistema integra:

  • Extracción de datos mediante API
  • Limpieza y transformación en Python
  • Almacenamiento en SQL Server
  • Visualización y análisis en Power BI
